# GitHub Pages for huangsam

Serves as a personal site for [huangsam](https://github.com/huangsam).

## Getting started

As simple as:

```bash
yarn install
yarn develop
```

The site is now running at `http://localhost:8000`.

## Deploying site

As simple as:

```bash
yarn deploy
```

The site is now deployed to the `gh-pages` branch.
