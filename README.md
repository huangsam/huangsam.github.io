## Getting started

Getting started is as simple as:

```bash
yarn install
yarn dev
```

The site is now running at <http://localhost:5173/>.

## Deploying site

Deploying the site is as simple as:

```bash
yarn deploy
```

The site is now deployed to the `gh-pages` branch.
